# Todo API

- get all todos
  (GET) `/api/v1/todos`
- create/add todo
  (POST) `/api/v1/todo`
- edit todo
  (PATCH) `/api/v1/todo/:id`
- delete todo
  (DELETE) `/api/v1/todo/:id`
